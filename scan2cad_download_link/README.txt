# Scan2CAD Dataset

Go to for proper description: https://github.com/skanti/Scan2CAD

## This folder contains:

- 'full_annotations.json' is the actual dataset with the Scan2CAD alignments and keypoints
- 'cad_appearances.json' is a helper file that contains how often a ShapeNet model appears per scene
- 'unique_cad.csv' is a helper file that contains a unique list of all used ShapeNet model in this dataset
